The print() statement.

Try out the print() statement by typing it into a file with .py extension (e.g. print_ex.py): print("Hello World!") Next run the file from the command prompt by typing (in Windows): python print_ex.py or (in OSX or Linux) python3.6 print_ex.py The 3.6 being the version of python installed on my system. Feel free to replace it with your version of python. The above is typed into the terminal from the level of the file. If you are currently in a different directory just type the path to the file instead of print_ex.py. It should output Hello World! into the console. When printing a string instance (actual text), and not a variable (more below), we place the string of characters in between quotation marks. print("Fast typing is fun.. ha ha.") or print('Hello You all.') The most common approach is to use single quotes. My personal preference though is to use double quotes to surround the string, so I can use single quotes within it without the need to escape them. print("I'd be happy if my dog wouldn't dream of a bowl full of my cat's food.") or print('She said "I live in Toronto."') This will cut the sentence before the letter 'I' and throw an error: print("She said "I don't live in Toronto"") We can escape any 'sensitive' character in a string with the backslash character: print("They told us \"We don't need no education\".")

Python3 format characters

The Unicode Standard is applied in Python 3 for representing any characters we type in the code. The reference of which is found here.

The below example demonstrates the usage of the old way of string formatting. For the other ways see New Way.

#!/usr/bin/env python                  
# -*- encoding: UTF-8 -*-
name = "Madej"
age = 36
height_cm = 170
height_inch = height_cm*0.39
weight_kg = 70
weight_lbs = weight_kg*2.2046
eyes = 2
fingers = "all intact"
hair = "curly"

print("If anyone is asking, my family's name is %s." %name)
print("The height of my body is %dcm which makes %d in inches." %(height_cm, height_inch)) 
print("Our Earth attracts me with the force of %xN, because I weigh %dkg or %dlbs if anyone prefers." %(weight_kg*9.81, weight_kg, weight_lbs))
print("I have %d eyes - as most of bigger species, and fingers %s. My head decoration is usually %s." %(eyes, fingers, hair))
Format Types & Their Character Representation
CharacterType
%b‒ binary
%c‒ integer to character (unicode) conversion
%d‒ decimal - number in base 10
%o‒ octal - number in base 8
%x‒ hex - number in base 16
%n‒ number - same as %d, but using the current locale setting for number separation.
%e‒ scientific notation with default precision of 6 digits after decimal point.
%E= %e
%f= fixed point notation with a given precision 'p'
%g‒ general format - with a given precision 'p'
%n‒ number
