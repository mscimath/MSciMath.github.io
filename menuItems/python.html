The print() statement.

Try out the print() statement by typing it into a file with .py extension (e.g. print_ex.py):
print("Hello World!")		

Next run the file from the command prompt by typing (in Windows):

python print_ex.py
 or (in OSX or Linux)
python3.6 print_ex.py 
 The 3.6 being the version of python installed on my system. Feel free to replace it with your version
 of python. 
 
 The above is typed into the terminal from the level of the file. If you are currently in a different 
 directory just type the path to the file instead of print_ex.py.

It should output Hello World! into the console.

When printing a string instance (actual text), and not a variable (more below),
we place the string of characters in between quotation marks.

print("Fast typing is fun.. ha ha.")

or 

print('Hello You all.')

The most common approach is to use single quotes. My personal preference though is to use
double quotes to surround the string, so I can use single quotes within it without the
need to escape them.

print("I'd be happy if my dog wouldn't dream of a bowl full of my cat's food.")

or

print('She said "I live in Toronto."')

This will cut the sentence before the letter 'I' and throw an error:

print("She said "I don't live in Toronto"")

We can escape any 'sensitive' character in a string with the backslash character:

print("They told us \"We don't need no education\".")

Thread of Execution and the Call Stack

Javascript is not a asynchronous language, 
it is a synchronous language with asynchronous 
capabilities.
JavaScript is a single-threaded language.
Thread is a single sequential flow of 
control - a process of a program:
  Thread of execution:
  op1 -> op2 -> op3 -> op4 -> Call Stack
                           -> Memory Heap

The Call Stack keeps track of functions (a stack 
of functions) to be executed & manages execution 
contexts. The Stacks are LIFO (last in first out).

  Call Stack :  third()
                second()
                first()
                Global EC (execution context)

A Stack is a data structure.

Example of a simple call stack:
                  function first() {
                    console.log('first...');
                  };
                  function second() {
                    console.log('second...');
                  };
                  function third() {
                    console.log('third...');
                  };
                  first();
                  second();
                  third();

                  output: first...
                          second...
                          third... 

Example of a just a little more complex call stack:
                  function first() {
                    console.log('first...');
                    second();
                  };
                  function second() {
                    console.log('second...');
                    third();
                  };
                  function third() {
                    console.log('third');
                  };
                  first();

                  output: first... 
                          second... 
                          third...

repeat()

Note: String.prototype.repeat() is an ES6 feature. It is an independent method as all string methods are, which means that it does not change the original string . It operates on code units.

Adding a comma after the start parameter of the .fill() method will apply the given value at all the indexies from the start to the end of the array.

var sameArr = new Array(7);
sameArr.fill('value', 0,)
console.log(sameArr);  //returns['value', 'value', 'value', 'value', 'value', 'value', 'value']
